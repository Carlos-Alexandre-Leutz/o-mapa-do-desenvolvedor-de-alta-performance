# 🚀 Sales Engine & Behavioral Analytics | Full-Stack MVP

### **Status: Active Production | Data-Driven Revenue**
This repository houses the core sales infrastructure for my digital products. Built with a **Lean Startup** philosophy, this project isn't just a landing page—it's a custom-built **Behavioral Analytics Engine** designed to optimize conversion through real-world user data.

---

## 📈 The Business Intelligence (BI) Edge
Instead of relying solely on third-party tools, I engineered a custom tracking system using **Firebase** to capture deep granular data:
* **Attribution Tracking:** Identifying exactly which YouTube video, Instagram post, or external link triggered the visit.
* **Engagement Metrics:** Measuring "Time to First Interaction" and specific user journey milestones.
* **Conversion Optimization:** Using data patterns to iterate on the UI/UX, directly influencing sales growth and product-market fit.
* **Revenue:** Currently generating approximately **R$ 2.000,00/month** in validated sales.

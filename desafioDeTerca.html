🚦 Desafio: Lógica do Semáforo

Objetivo: Criar uma lógica que diga ao motorista o que fazer dependendo da cor do sinal.

⚠️ A Missão:
Se a cor for "verde", o resultado deve ser "Siga em frente".
Se for "amarelo", "Atenção: Reduza".
Se for "vermelho", "Pare o veículo".

💡 Dica: Use else if (cor === "amarelo") para verificar a segunda condição.

📘 Explicação

Na programação, usamos o IF (SE) para tomar decisões. Quando temos mais de duas opções, usamos o ELSE IF.

if (cor === "verde") {
  acao = "Siga";
} else if (cor === "amarelo") {
  acao = "Atenção";
} else {
  acao = "Pare";
}

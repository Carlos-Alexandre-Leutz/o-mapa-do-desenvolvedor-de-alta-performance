🐞 Desafio: Bug da Tipagem

Objetivo: Corrigir o código para que ele realize uma soma matemática em vez de apenas "colar" os textos.

⚠️ O Bug:
O sistema deve prever sua idade daqui a 5 anos. Mas se você digita 20, ele diz que você terá 205!

Digite sua idade atual:

Código JavaScript:

💡 Dica: No JS, "20" + 5 = "205". Use Number() para converter.

📘 Explicação

Tudo o que o usuário digita no navegador é lido como texto (String). No JS, o sinal de + com texto serve para juntar (concatenar).

Para somar matematicamente, você deve converter o texto em número:

let idadeAtual = Number(valorDigitado);
